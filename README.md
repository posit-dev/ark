# Amalthea

## What is it?

R Kernel for Jupyter and Myriac, written in Rust. 

## Features

None yet. Coming soon!

### What's with the name?

This is a Jupyter kernel; Amalthea is [one of Jupiter's moons](https://en.wikipedia.org/wiki/Amalthea_(moon)).
